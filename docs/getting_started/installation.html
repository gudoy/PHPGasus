Vhost or .htaccess

  1. add vhost to your server conf or add the proper .htaccess at the root of your projet
  2. edit conf file if necessary (change _PATH & _URL)
  3. edit APP_CONTEXT env var with proper value (local/dev/preprod/prod)

Set permissions

Attention : Depending of who owns the folders of your project, you may have to set higher rights (770, 775 or even 777) for the following folders

  • templates/_precompiled/ 755
  • [templates/_cache/ 755]] (required if you want to use templates caching)
  • [templates/_minified/ 755] (required if you want to use minified files caching)
